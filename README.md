# Live Character Counter

A modern **React-based web application** that counts characters and words in real time as the user types.
The application provides instant feedback on the number of characters, words, and remaining characters within a specified limit.

---

## 🚀 Features

* ✨ **Real-time Character Counting** – Instantly updates the number of characters as you type.
* 📝 **Word Counter** – Tracks the number of words in the textarea.
* 📊 **Remaining Character Limit** – Shows how many characters are left out of the maximum limit.
* ⚠️ **Limit Warning** – Displays a warning when the character limit is exceeded.
* 🔄 **Reset Button** – Clears the textarea and resets all counters.
* 🎨 **Clean and Responsive UI** – Modern design with gradient background and card layout.
* ⚡ **Fast Performance** – Built with React and Vite for quick rendering and development.

---

## ▶️ How to Run the Project

### 1. Clone the Repository

```
git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git
```

### 2. Navigate to the Project Folder

```
cd character-counter
```

### 3. Install Dependencies

```
npm install
```

### 4. Start the Development Server

```
npm run dev
```

Open the browser and go to:

```
http://localhost:5173
```
