## Features Explained

### Character Counter
Displays the total number of characters typed in the textarea (max 500 characters).

### Word Counter
Automatically counts the number of words based on whitespace separation.

### Remaining Characters
Shows how many characters are left before reaching the 500 character limit.

### Progress Bar
Visual representation of text usage with a filled gradient bar.

### Clear Button
Clears all text with a single click for quick reset.

### Responsive Design
Adapts beautifully to different screen sizes including mobile devices.
